# Golf Croquet

Tactics, strategy, and rules for Golf Croquet. The game runs one ball per turn — first to run six hoops wins. Simple to start, deep to master.

## Game analysis — Cowing

The **Cowing method** is a structured approach to game analysis — tracking shot outcomes, patterns, and decision points across a match.

The vault has an extended dataset from Addison Matthews' Gold Medal Tournament games:

- [[resources/cowing analysis/Cowing Analysis — Addison Matthews (Extended Dataset + Sheet)\|Extended Dataset Analysis]]
